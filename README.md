# Exploratory Data Analysis & Business Intelligence

## Objective
Analyze sales data to uncover trends, patterns, and business insights using Python, SQL, and Power BI.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- MySQL
- Power BI

## Project Workflow
1. Data Cleaning
2. Descriptive Statistics
3. Univariate Analysis
4. SQL Business Queries
5. Multivariate Analysis
6. Dashboard Development

## Key Insights
- Electronics generated the highest revenue.
- Mobile was the highest revenue-generating product.
- Patna contributed the highest city revenue.
- Customers aged 55+ generated the highest revenue.
- Revenue peaked during March and June 2025.
- Grocery had the highest average order value.

## Deliverables
- EDA Notebook
- SQL Queries
- Power BI Dashboard
- Business Insights Report
